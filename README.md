My engineering background spans AWS, Kubernetes, ECS Fargate, Terraform, Docker, Helm, Jenkins, GitHub Actions, Azure DevOps, Prometheus, Grafana, Elasticsearch, Ruby on Rails, Golang, Python, and modern AI/ML infrastructure.
